	```java
	public void printTwos(int n) {
		if (n < 1) {
			throw new IllegalArgumentException();
		}
		
		if (n % 2 == 1) {
			System.out.print(n);
		} else if ((n / 2) % 2 == 1) {
			System.out.print("2 * ");
			printTwos(n / 2);
		} else {
			System.out.print("2 * ");
			printTwos(n / 4);
			System.out.print(" * 2");
		}
	}
	```

1. Array Programming, 10 points. 

	```java
	public int removeLast(int n) {
		int index = -1;
		for (int i = 0; i < size; i++) {
			if (elementData[i] == n) {
				index = i;
			}
		}

		if (index == -1) {
			return -1;
		} else {
			for (int i = index; i < size - 1; i++) {
				elementData[i] = elementData[i + 1];
			}
			size -= 1;
			return index;
		}
	}
	```

1. Stacks/Queues, 25 points. 

	```java
	public int removeMin(Stack<Integer> s) {
		Queue<Integer> q = new LinkedList<Integer>();  
		int min = s.pop();
		q.add(min);
		while( !s.isEmpty()) {
			int next = s.pop();
			if (next < min) {
				min = next;
			}
			q.add(next);
		}
		while (!q.isEmpty()) {
			int next = q.remove();
			if (next != min) {
				s.push(next);
			}
		}
		while (!s.isEmpty()) {
			q.add(s.pop());
		}
		while (!q.isEmpty()) {
			s.push(q.remove());
		}
		return min;
	}
	```
